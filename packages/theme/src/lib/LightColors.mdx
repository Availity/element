# Colors

Colors for `light` theme.

Listed colors are directly available through the `sx` property for select properties (see [MUI System Properties Documentation](https://mui.com/system/properties/))

e.g. `sx={{ bgcolor: 'primary.main' }}`

<ColorPalette>
  {Object.entries(lightTheme.palette).map(([name, value]) => {
    let values= typeof value === 'object' ? value : {[name]: value};
    values = Object.fromEntries(Object.entries(values).filter(([k,v]) => isNaN(v)));
    if (JSON.stringify(values) === '{}') return;
    return(
      <ColorItem
        title={name}
        colors={{...values}}
      />
  )})}
</ColorPalette>
